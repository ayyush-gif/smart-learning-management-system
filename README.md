# 🎓 Lumina LMS

A modern full-stack **Learning Management System (LMS)** built using the **MERN Stack**. Lumina enables students to discover and enroll in courses, track their learning journey, and download completion certificates, while instructors can manage courses through a dedicated dashboard.

---

## 🌐 Live Demo

**Frontend:** https://smart-learning-management-system-plum.vercel.app/

---

## ✨ Key Features

### 👨‍🎓 Student Features

* Secure User Registration & Login
* JWT Authentication & Authorization
* Browse Available Courses
* Course Search Functionality
* Enroll in Courses
* Personalized Student Dashboard
* User Profile Page
* Certificate Generation
* PDF Certificate Download

### 👨‍🏫 Instructor Features

* Create New Courses
* View All Courses
* Edit Existing Courses
* Delete Courses
* Manage Course Information

### 🚀 Platform Features

* Modern Responsive UI
* Role-Based Access Control
* RESTful API Architecture
* MongoDB Database Integration
* Secure Authentication System
* Cloud Deployment
